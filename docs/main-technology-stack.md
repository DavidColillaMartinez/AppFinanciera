# Main Technology Stack

## Framework

- Next.js con App Router.
- React.
- TypeScript estricto.

## UI

- TailwindCSS.
- shadcn/ui.
- Radix UI primitives.
- Lucide React.
- Recharts.

## Estado y Datos

- Zustand para estado local simple.
- TanStack Query para lecturas, cache, refresco e invalidacion de datos remotos.
- React Hook Form para formularios.
- Zod para validacion y contratos.

## Google

- Google OAuth con permisos de usuario.
- Google Sheets API como acceso a datos.
- La app debe conectarse a una Sheet copiada por cada usuario.

## PWA

- Manifest web.
- Metadatos compatibles con instalacion desde Safari.
- `@serwist/next` queda disponible para service worker cuando se active la fase PWA completa.

## Versionado

- Git desde el inicio del desarrollo real.
- Commits pequenos y especificos.
